BDC Messenger

Realtime Full-Stack Chat Application

Author: Mohd Saad Mohd Shakeel — Course: Bachlor Of Computer Application — Short demo & architecture overview

Use ← → or click to navigate. Press Esc to jump to title.

Problem & Objectives

  • Build a realtime chat with persistence and presence.
  • Secure authentication and profile management.
  • Responsive UI and smooth UX for messaging.

Key Features

  • Signup / Login / Profile updates
  • One-to-one messaging with images
  • Realtime updates (Socket.io) and online users
  • Optional call signaling using WebRTC

Architecture & Tech Stack

Frontend: React (Vite), TailwindCSS, DaisyUI, Zustand, Axios

Backend: Node.js (ESM), Express, Socket.io, Mongoose, MongoDB Atlas

Media: Cloudinary for images

How it works (flow)

  1. Client calls REST API for auth and messages.
  2. Client connects to Socket.io for realtime events.
  3. Server persists messages in MongoDB; broadcasts new messages via Socket.io.

Run & Demo

Install and run:

npm install --prefix backend
npm install --prefix frontend
cd backend
npm start
cd ../frontend
npm run dev
Open http://localhost:5173
